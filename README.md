# 🟡 Blinkit Sales Dashboard — Power BI
![Dashboard](Blinkit%20Das%20board.png)

## 📊 Overview
An interactive Sales Analytics Dashboard built in Power BI, inspired by Blinkit's real-world data structure. Designed to simulate how quick-commerce brands track outlet performance and product trends.

## ✨ Features
- Total Sales & Revenue breakdown
- Item-wise & Category-wise performance analysis
- Outlet size & location-based insights
- Dynamic filter panel for interactive exploration
- Custom DAX measures for KPI calculations

## 🛠 Tools & Technologies
| Tool | Usage |
|------|-------|
| Power BI | Dashboard building & visualization |
| DAX | Custom measures & calculations |
| Data Modeling | Table relationships & schema design |
| Power Query | Data cleaning & transformation |
